if ~ else 문제, 파이썬은 if, elif, else로 이루어져있다.

사용자가 값을 입력할 때에

```python
input()
```

은 모든 것을 <b>문자열</b>로 취급한다.

따라서 자료형을 변경할 수 있다.

```python
int(input()) # 정수형
float(input()) # 실수형
```
